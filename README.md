<h1 align="center"> :small_red_triangle: Simple Nest.js CRUD</h1>

## Description

Simple Node.js App Deploy to EC2

## :gear: Technologies

- Swagger
- Nest.js
- AWS EC2
- MongoDB
- TypeScript

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
